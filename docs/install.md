# Install Percona Binary Log Server

The latest version is available on Docker.

Use Docker when a ready-to-run deployment is preferred. Use a source build when local compilation, debugging, packaging work, or source-code changes are required.

## Choose An Installation Method

* using Docker for the latest packaged version

* building from [source](build-from-source.md) for local compilation and development work
